How to Run
Install the required library:
py -m pip install google-genai
Create your own Gemini API key from Google AI Studio.
Set your API key in PowerShell:
$env:GEMINI_API_KEY="YOUR_API_KEY_HERE"

Replace YOUR_API_KEY_HERE with your own Gemini API key.

Run the chatbot:
py chatbot.py

Note: The API key is not included in this repository for security reasons. Each user must use their own Gemini API key.
